# AI Development Prompts Used

## 1. Project Planning Prompt

Create a full-stack car dealership management system using React for frontend and Django REST Framework for backend. The system should allow users to view vehicles, register/login, and purchase cars. Admin users should manage vehicles and track purchases.

---

## 2. Backend Development Prompt

Build a Django REST Framework backend for a car dealership system with:
- Vehicle model containing brand, model, year, price, and stock
- REST APIs for listing vehicles, viewing details, and purchasing vehicles
- JWT authentication
- Stock reduction after successful purchase

---

## 3. Frontend Development Prompt

Develop a React frontend for the car dealership system with:
- User authentication
- Vehicle listing cards
- Vehicle details page
- Purchase functionality
- Protected routes
- Admin dashboard

---

## 4. Debugging Prompt

Help debug errors in React and Django integration, including API connection issues, authentication problems, CORS errors, and frontend-backend communication.

---

## 5. Documentation Prompt

Generate project documentation including:
- Features
- Technology stack
- API details
- Testing information
- Future improvements
